基于 anaconda

anaconda

conda create -n rumor1 python=3.6 torch==1.7 torchtext==0.8 tqdm=4.19

conda info -e （查看所有的虚拟环境）

1.创建环境
conda create -n test python=3.6 # test为环境名，3.5为python的版本号

2.删除环境
conda remove -n test –all

3.环境重命名
conda create -n tf –clone rcnn #把环境 rcnn 重命名成 tf
conda remove -n rcnn –all
#conda 其实没有重命名指令，实现重命名是通过 clone 完成的，分两步：1.先 clone 一份 new name 的环境。2.删除 old name 的环境。

5将环境写入notebook的kernel中

python -m ipykernel install –user –name liuzhch

6使用命令jupyter kernelspec remove 'kernelname'可以删除指定的kernel

7使用命令jupyter kernelspec list可以查看jupyter所有的kernel

退出环境 conda deactivate

source activate
